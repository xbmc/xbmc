Mbedthis Appweb Roadmap

The Appweb roadmap is created by the accumulated input from all Appweb users. As such, it is subject to change. This roadmap is not a commitment to product, but the current thinking about future directions.

Those who shout loudest -- often get what they ask for. To request a feature, please send email to:

dev@mbedthis.com

Feature Wish List

This list is not in any particular order.
  • ASP.NET module (underway)
  • Python module
  • Perl module
  • Administration Console
  • Library of Graphic Applets for use in a Browser
  • SSL support for httpClient
  • Compressed ROM pages
  • SNMP manageability

What's in the Next Release

The next release in the stable branch will be the Mbedthis Appweb 2.2 which is currently unscheduled. Patches to the stable branch are released only as issues require it.

For the development branch, we will release incremental technology updates and fixes. The next is 3.0 which is expected in 2H 2006.

Please tell us what you would like to see at dev@mbedthis.com.
